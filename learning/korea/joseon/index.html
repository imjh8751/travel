조선왕조 쉽게 외우기 👑

왕 이름만 줄줄 외우지 말고, 왜 중요한지, 무슨 사건이 이어졌는지, 어떤 흐름으로 바뀌는지 그림처럼 이해하도록 만든 학습용 HTML입니다.

태정태세문단세 성리학 국가 세종=과학·훈민정음 임진왜란·병자호란 영정조 르네상스 세도정치→개항→멸망

1. 한 문장으로 전체 흐름 잡기

건국 → 제도 정비 → 문화 황금기 → 사화와 붕당 → 전쟁 극복 → 탕평과 개혁 → 세도정치 → 개항과 몰락
  • 초기: 나라를 세우고 제도 틀을 만든 시기
  • 중기: 사림 성장, 사화, 붕당으로 정치 갈등 심화
  • 후기: 임진왜란·병자호란 이후 재건, 영정조 개혁
  • 말기: 세도정치, 민란, 외세 침략, 개항, 대한제국으로 연결
그림 해석: 금색(건국) → 초록(문화 발전) → 파랑(전쟁·위기 극복) → 이후 말기 쇠퇴

3. 시기별 핵심 그림 설명

🏯 건국기

태조~태종은 새 집을 짓는 단계입니다. 나라를 세우고, 왕권을 강화하고, 수도를 한양으로 정비합니다.

📚 황금기

세종~성종은 집 안에 책, 악기, 과학도구를 채우는 단계입니다. 훈민정음, 과학, 법과 제도가 완성됩니다.

⚔️ 혼란기

연산군~선조는 집안 싸움과 외부 침입이 겹친 단계입니다. 사화와 붕당, 임진왜란이 이어집니다.

🛡️ 재건기

광해군~현종은 무너진 집을 수리하는 단계입니다. 전후 복구, 북벌론, 호란 대응이 핵심입니다.

🌿 부흥기

숙종~정조는 다시 질서를 잡고 공부와 개혁을 시도한 단계입니다. 탕평책, 실학, 규장각이 중요합니다.

🌧️ 몰락기

순조~순종은 권문세가 장악, 민란, 외세 침략, 개항으로 왕조가 기울어지는 단계입니다.

6. 진짜 쉽게 외우는 포인트

세종

“글자 만든 왕” 하나만 기억하면 안 됩니다. 훈민정음 + 과학 + 북방영토 + 농업까지 묶어서 외우세요.

선조

“임진왜란 때 왕”. 붕당 심화와 전쟁이 겹친 시기라는 흐름으로 기억하세요.

영조·정조

둘은 묶어서 외우면 쉽습니다. 탕평(영조) → 개혁과 문화(정조).

광해군

“나쁜 왕/좋은 왕” 단순평가보다 전후복구 + 대동법 + 중립외교로 이해하는 게 중요합니다.

고종

개항기 핵심입니다. 외세 침략 + 근대개혁 + 대한제국으로 기억하면 정리가 쉽습니다.

말기 전체

세도정치 → 민란 → 외세 침략 → 개항 → 대한제국 → 병합 순서로 큰 줄기를 잡으세요.
